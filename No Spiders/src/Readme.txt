SUMMARY
=======

Thief Gold and Thief II no-spider patch. Version 2.0, April 12 2018.

Created by Andy Durdin (me@andy.durdin.net)

Tested with Thief Gold 1.25 NewDark (GOG.com release) and Thief II
1.25 NewDark (GOG.com release). If you have any problems with it on
other versions, let me know.

Please feel free to redistribute this patch.


OVERVIEW
========

This patch makes all the spiders:

  a) friendly to the player
  b) attacked by other ai
  c) have only one hit point
  d) die as soon as the level starts anyway
  e) look like a small white mossy stone
  f) invisible
  g) teleported out of the level.

This applies to all: small spiders, big spiders, chaos spiders, and
spiderbots.


INSTALLING
==========

1. In your Thief Gold or Thief 2 directory, create a new subdirectory
   called "dbmods".

2. Copy the corresponding gamesys.dml file into that subdirectory.

3. Run the game.


UNINSTALLING
============

1. Delete the dbmods\gamesys.dml file


Have fun!
